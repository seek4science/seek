<a name="isa_tags"></a>**ISA tags** are used to define where the different experimental assets need to end in the ISA-JSON.

The  different ISA-tags are directly linked to Sample Attirbutes and differ based on the sample type's level:

- Study `source` level sample type:
  - `source`
  - `source_characteristic`
- Study `sample collection` level sample type:
  - `sample`
  - `sample_characteristic`
  - `protocol`
  - `parameter_value`
- `other_material` level sample type:
  - `other_material`
  - `other_material_characteristic`
  - `protocol`
  - `parameter_value`
- `data file` level sample type:
  - `data_file`
  - `data_file_comment`
  - `protocal`
  - `parameter_value`
